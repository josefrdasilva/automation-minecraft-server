# Minecraft Server Infrastructure
### Projeto de infraestrutura para servidor Minecraft com automação de backups e gestão via Docker.
## Tecnologias

    OS: Ubuntu Server 24.04

    Container: Docker & Docker Compose

    Automação: Shell Scripting & Cron

    Networking: Playit.gg

## Funcionalidades

    Automação: Backup automatizado de hora em hora.

    Gestão: Comandos via CLI para chat e logs do servidor.

    Segurança: Configuração de permissões de usuário (ID 1000) e gestão de containers.
